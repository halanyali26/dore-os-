# Dore OS Dashboard Design System

## Palette
- Background: `#0c0c0c`
- Panel: `#141414`
- Border: `#222`
- Text: `#b0b0b0`
- Accent (amber): `#d4842a`
- Green: `#3a8`
- Dim: `#555`

## Typography
- Font: JetBrains Mono → Menlo → Consolas → monospace
- Size: 12px body, 22px headings
- Uppercase, letter-spacing: 2-4px

## Effects
- CRT scanlines: `repeating-linear-gradient` at 4px intervals
- Vignette: `radial-gradient` overlay
- State badges: bordered outlines (no filled backgrounds)
- Hover: amber underline on stat cards

## NEVER Use
- Purple gradients
- Inter/Roboto/system fonts
- Rounded corners
- Pastel colors
- Generic AI aesthetic

## Reference
Anthropic frontend-design skill principles applied.
